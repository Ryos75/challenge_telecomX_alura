# challenge_telecomX_alura
Este projeto tem como objetivo analisar a evasão de clientes (churn) da empresa fictícia TelecomX , utilizando técnicas de Análise Exploratória de Dados (EDA) . A partir dos dados disponibilizados, busque identificar padrões e fatores que influenciam o cancelamento dos serviços, apoiando uma tomada de decisões estratégicas para retenção de clientes
#Objetivo

Compreender o comportamento dos clientes em relação à evasão

identificar perfis com maior risco de churn

Explorar variáveis ​​categóricas e numéricas associadas ao cancelamento

Gerar insights e recomendações baseadas em dados
